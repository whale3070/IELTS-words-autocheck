确保在同一目录下有 task数字.txt  系列的文件，并在其中包含正确的单词列表和中文解析。在命令行中运行 "python3 spell_check.py"，然后根据提示输入一个任务文件的数字（例如32）。接下来，输入单词，一行一个。按 q 结束输入，程序将输出所有拼写错误的单词、正确的单词用法以及正确单词的行号，并计算正确率。
